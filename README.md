**RoCall** is a Python/Discord bot that allows Roblox games to **interact with Discord voice channels**. It provides a way for users to manage voice interactions from Roblox in **controlled, private environments**.

---

## ⚠️ Important Notice

- RoCall is intended for **invite-only or private use**. Using it in public games or servers **may violate Roblox Terms of Service**.  
- The tool is provided for **educational and development purposes**, and users are responsible for ensuring their implementation follows Roblox and Discord rules

## Key Features

- **Discord voice channel integration** from Roblox  
- Supports **voice management in private environments**  
- Example Roblox code for easy integration  
- Fully open-source and hosted on GitHub  

RoCall enables users to create **interactive, voice-enabled Roblox experiences in an alternative way**.



## Setup

1. Add [RoCall](https://discord.com/oauth2/authorize?client_id=1451650442487402678) and [Bloxlink](https://discord.com/oauth2/authorize?client_id=426537812993638400&permissions=469887038&scope=bot+applications.commands) to your Discord server
2. Configure RoCall using built-in commands (main vc, idle vc, password, and [Bloxlink API token](https://blox.link/dashboard/user/developer))
3. [Download Roblox model](https://github.com/Ha1wen/RoCall/raw/main/RoCall.rbxm) and put it in ServerScriptService
4. Edit the Config module inside the model

Current public API (may experience delays): https://rocall.onrender.com/api
